# Java Ecosystem and Fundamentals

Java was originally created by James Gosling and his team at Sun Microsystems. Currently, the company that owns Java is Oracle Corporation.

> [!info] Does Java cost money?
> It depends on the version you use. 
> - **OpenJDK:** It is the open-source implementation and is 100% free.
> - **Oracle JDK:** It is the commercial version. Although it includes the OpenJDK code, it may offer specialized support and additional tools, but requires a subscription.

## Language Structure
The Java ecosystem is mainly divided into levels of complexity:
- **Java SE (Standard Edition):** It is the base of the language and includes basic classes, threads, and file handling.
- **Jakarta EE (formerly Java EE):** These are the extensions designed for enterprises, providing tools for web servers, messaging, and databases.

## Industry Adoption
In the business world, the most widely used versions currently are the **LTS (Long Term Support)** ones, with **Java 17** being particularly prominent. 

Large companies worldwide rely on Java for their **[[Arquitectura Base de Software y Web#Frontend vs Backend|Backend]]** architecture, including:
- Netflix 
- Spotify (which also offers a [[Arquitectura Base de Software y Web#APIs (Application Programming Interface)|public API]])
- Uber 
- LinkedIn 
- Minecraft 

To develop and deploy these applications, different **[[Software Environments]]** (Development, Staging, Production) are used.

To develop in this language, programmers use an **IDE (Integrated Development Environment)**, which is software that includes a text editor, debugging tools, and a compiler. Most modern projects rely on **[[Frameworks de programación y Spring|Frameworks]]** to accelerate development.
