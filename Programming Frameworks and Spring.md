# Programming Frameworks

A framework is a set of tools and predefined rules that help programmers avoid writing everything from scratch. 

## Frameworks by Language
The main programming frameworks are as follows:
- **[[Java - History and Ecosystem|Java]]:** Spring, Quarkus.
- **Python:** Django, Flask.
- **JavaScript (JS):** React, NestJS.
- **PHP:** Laravel.

---

## The Spring Ecosystem in Java
**Spring** is Java's most popular framework for creating enterprise and **[[Core Software and Web Architecture#Frontend vs Backend|Backend]]** applications. It is owned by Broadcom.

- **Main purpose:** Simplify development, allowing the programmer to focus on business logic instead of complex server configuration.
- **Environment Management:** Facilitates configuration for different **[[Software Environments|environments]]** (for example, using *Spring Profiles* to separate Dev from Prod).
- **Competition:** Competes with Jakarta EE by being lighter and more flexible.

### Alternatives to Spring
There are other frameworks with specific advantages:
1. **Quarkus:** Optimized for the cloud and containers.
2. **Micronaut:** Ideal for microservices due to its low memory consumption.
3. **Jakarta EE:** It is the official standard alternative, very stable for banking.
